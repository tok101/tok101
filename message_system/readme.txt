私有库依赖：
	paho.mqtt_encapsulate
	LinkList

功能：
	这个库提供了一种模块间消息通讯的机制。用户模块可以通过这个库加入到“消息系统”，之后便能够接收到同一系统中其他模块发送的消息，或者向其他模块发送消息。

说明：
	现有两种类型的消息：
	1.广播：
		某个模块向系统广播某事件，系统中所有加入的模块都可以接受到此事件的广播消息。这些模块可以通过预先设定的处理函数处理该事件。例如关机事件。
	2.p2p：
		某个模块向系统发送某p2p消息，如果系统中有模块可以回应该消息，那么该模块将会响应该消息，并把结果返回给发送模块。

编译
	运行make

测试：
	1.运行mosquitto：mosquitto -v
	2.运行测试程序
